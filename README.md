## Terminos de Hardware
### `CPU` — Unidad Central de Procesamiento
Es el cerebro del computador. Procesa todas las instrucciones
que le damos al sistema. Sin el, ninguna operacion es posible.
### `RAM` — Memoria de Acceso Aleatorio
Es la memoria temporal del computador. Guarda lo que estas
usando en este momento. Al apagar el computador, se borra.
### `SSD` — Disco de Estado Solido
Es donde se guarda la informacion de forma permanente.
Es mas rapido que un disco duro tradicional (HDD).

## Terminos de Software
*(
### `Base de datos` - Una base de datos es un sistema diseñado para almacenar grandes cantidades de información de forma organizada, permitiendo consultarla, modificarla y gestionarla fácilmente. Se utilizan en casi todas las aplicaciones modernas, como redes sociales, sistemas bancarios o tiendas en línea. Existen diferentes tipos, como bases de datos relacionales (que organizan los datos en tablas) y no relacionales (más flexibles). Su objetivo principal es garantizar que los datos sean accesibles, seguros y consistentes.

### `API` - Una API es un conjunto de reglas y protocolos que permite que distintos programas se comuniquen entre sí. Funciona como un intermediario que facilita el intercambio de información sin que los desarrolladores tengan que conocer todos los detalles internos del sistema con el que interactúan. Por ejemplo, cuando una aplicación muestra el clima, probablemente está usando una API externa para obtener esos datos. Las APIs son fundamentales en el desarrollo moderno porque permiten integrar servicios de forma rápida y eficiente.

### `Framework` - Un framework es una estructura predefinida que proporciona herramientas, librerías y buenas prácticas para desarrollar software. Su propósito es simplificar el trabajo del programador, evitando que tenga que empezar desde cero. Incluye componentes reutilizables y una forma organizada de construir aplicaciones. Por ejemplo, hay frameworks para desarrollo web, móvil o incluso inteligencia artificial. Usar un framework ayuda a mantener el código más limpio, escalable y fácil de mantener.)*
